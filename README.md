# Electron & React Typescript
## Using Nest.js
https://nestjs.com/

## Getting started
Clone the repo
```
git clone https://github.com/ASAPHAANING/electron-react-nestjs.git
```
Install dependencies
```
yarn
```
Package the app `yarn build:prod` then run it `yarn start`
